# 🐍 Fundamentos de Python

Repositório criado para registrar meus estudos iniciais em Python por meio de exercícios práticos desenvolvidos durante o aprendizado da linguagem.

## 📚 Conteúdos praticados

- Variáveis e tipos de dados
- Entrada e saída de informações
- Operações matemáticas
- Estruturas condicionais (`if`, `elif`, `else`)
- Operadores lógicos
- Laços de repetição (`for` e `while`)
- Funções
- Listas e dicionários
- Manipulação de dados em memória
- Pequenos programas executados no terminal

## 🎯 Objetivo

Consolidar os fundamentos da programação em Python através da prática, desenvolvendo a lógica de programação e a compreensão dos principais recursos da linguagem.
