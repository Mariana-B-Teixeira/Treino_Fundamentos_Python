# 📖 Sobre o curso

O **CS50's Introduction to Programming with Python (CS50P)** é um curso introdutório de programação da Harvard University, ministrado por **David J. Malan**.

Ao longo do curso, são apresentados os principais fundamentos da programação utilizando exclusivamente a linguagem **Python**, por meio de aulas práticas, exercícios e desafios que estimulam o raciocínio lógico e a resolução de problemas.

Este repositório tem como objetivo registrar minha evolução durante o curso e reunir todas as soluções desenvolvidas ao longo dessa jornada.
